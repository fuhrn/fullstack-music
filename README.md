## FullStack Music - Track

Se trata de una aplicación simil Spotify en la que el usuario tiene listas de temas elegidos y luego puede ejecutar cada canción y recorrer el tema con botones de parar, arrancar, avance hacia adelante y hacia atras, etc

Esta aplicación esta hecha con Next.js (React), ChakraUI, PostgreSQL con Prisma sobre Heroku. Esta aplicación es el resultado de un curso hecho en FrontedMasters. Deployada en Vercel.

NOTA: para ver el funcionamiento de la aplicación, ingresar el usuario y clave de prueba indicado en la landing page.

La aplicación está deployada en: https://fuhr-nestor-fullstack-music.vercel.app
